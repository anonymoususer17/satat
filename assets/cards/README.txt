Card Image Naming Convention
=============================

Each card PNG should be named using the format: {suit}_{rank}.png

Examples:
- hearts_ace.png (Ace of Hearts - the one you already have!)
- hearts_two.png (2 of Hearts - the special H2 card)
- diamonds_king.png
- clubs_three.png
- spades_queen.png
